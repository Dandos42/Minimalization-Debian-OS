# Minimizing the Debian OS

## Description
Minimizing the size of the Debian operating system while still being able to view the page www.vut.cz and display the SOS subject script
* Original size: 3GB
* Final size: 0,88 GB

The resulting .os.ova file cannot be uploaded due to its size
